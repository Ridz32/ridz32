Ligne de commande Description Comment utiliser
cd Change le répertoire ex. "cd mon-nouveau-répertoire"
ls affiche la liste des fichiers du répertoire ex. "ls"
chmod gère les permissions d'accessibilité d'un fichier ex. "chmod u=rwx,g=rx,o=r myfile.ext"
chown change qui peut accéder à un fichier ex. "chown username myfile.ext"
